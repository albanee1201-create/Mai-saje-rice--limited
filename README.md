# 🌍 Overview

MAI - SAJE LIMITED is a modern agricultural and technology-driven rice company based in Nigeria.
This project is designed to build a complete digital ecosystem for rice production, processing, packaging, distribution, inventory management, farmer partnerships, and online sales.

The platform combines agriculture, logistics, finance, cloud technology, and smart business management into one integrated system.

# ⚙️ Installation Guide

## Clone Repository

```bash
git clone https://github.com/your-username/mai-saje-limited.git
```

## Navigate to Project

```bash
cd mai-saje-limited
```

---

# 📥 Frontend Installation

```bash
cd frontend
npm install
npm run dev
```

---

# 📥 Backend Installation

```bash
cd backend
npm install
npm run server
```

---

# 📱 Mobile App Installation

```bash
cd mobile-app
flutter pub get
flutter run
```

---

# 🗄️ Database Setup

Configure your environment variables:

```env
DATABASE_URL=
JWT_SECRET=
PAYSTACK_SECRET=
FLUTTERWAVE_SECRET=
```

Run database migration:

```bash
npm run migrate
```

---

# 🔑 Environment Variables

Create a `.env` file:

```env
PORT=5000
NODE_ENV=development

DATABASE_URL=your_database_url

JWT_SECRET=your_secret_key

PAYSTACK_SECRET_KEY=your_key
FLUTTERWAVE_SECRET_KEY=your_key

CLOUDINARY_API_KEY=your_key
AWS_ACCESS_KEY=your_key
```
